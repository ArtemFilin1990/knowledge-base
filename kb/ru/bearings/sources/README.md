# Источники данных по подшипникам

Эта папка содержит оригинальные файлы-источники: каталоги производителей, прайс-листы, справочники.

## Назначение

- Хранение исходных материалов в неизменном виде
- Обеспечение прослеживаемости данных
- Архив для проверки и аудита

## Структура

```
_sources/
├── catalogs/           # Каталоги производителей (PDF, XLSX)
│   └── YYYY/           # По годам
│       ├── SKF_Catalog_YYYY.pdf
│       ├── FAG_Catalog_YYYY.pdf
│       └── ...
├── price-lists/        # Прайс-листы поставщиков
│   └── YYYY/           # По годам
│       └── supplier/   # По поставщикам
│           └── price_YYYY-MM.xlsx
├── standards/          # Копии/выдержки стандартов
│   ├── ISO_15_summary.pdf
│   ├── ISO_281_summary.pdf
│   └── GOST_520-2011_extract.pdf
└── technical-docs/     # Техническая документация
    ├── installation-guides/
    └── maintenance-manuals/
```

## Правила работы с источниками

1. **Не изменять оригиналы** — файлы хранятся «как есть»
2. **Именование**: `{производитель}_{тип}_{год}.{ext}`
3. **Версионирование**: при обновлении сохранять старую версию
4. **Метаданные**: создавать README с описанием содержимого
5. **Крупные файлы**: при необходимости использовать Git LFS или внешнее хранилище

## Обработка

Исходные файлы обрабатываются и преобразуются в структурированные данные в папке `datasets/`.
