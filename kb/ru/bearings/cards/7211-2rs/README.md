# 7211-2RS — подшипник шариковый радиально-упорный

Шариковый радиально-упорный однорядный подшипник с углом контакта 40° с двухсторонним резиновым уплотнением. Подшипник серии 7xxx, применяемый в промышленном оборудовании и механизмах.

## Назначение и где применяется

**Типовые применения**:
- Шпиндели металлорежущих станков
- Редукторы с коническими и червячными передачами
- Насосы высокого давления
- Приводы с комбинированной радиально-осевой нагрузкой

**Тип нагрузки**:
- Радиальная: основная составляющая
- Осевая: значительная, в одном направлении (до 60% от C)

**Режимы работы**:
- Обороты: 480–4800 об/мин (оптимум 480–1600)
- Температура: –30…+120 °C (с учётом смазки)
- Среда: чистая или умеренно загрязнённая (открытая конструкция — требуется внешнее уплотнение)

## Расшифровка обозначения

**Базовое обозначение**: 7211
- **7** — серия шариковых радиально-упорных (angular contact)
- **2** — серия ширины лёгкая (B = 21.0 мм для этого размера)
- **11** — код внутреннего диаметра: d = 11 × 5 = 55 мм

**Суффиксы**:
- **2RS** — двойное резиновое уплотнение (контактное, с двух сторон)
  - Защита от пыли, грязи, воды
  - Снижает предельные обороты (vs открытый или 2Z)
  - Пластичная смазка заложена на весь срок службы

**Особенности маркировки**:
- SKF: 7211-2RSH (H = Heavy duty уплотнение)
- FAG: 7211.2RSR (точка вместо дефиса)
- NTN: 7211LLU (LLU = низкое трение)
- NSK: 7211DDU (DDU = двойное уплотнение)

## Габариты и совместимость

**Размеры** (ISO 15):
- Внутренний диаметр d: 55 мм
- Наружный диаметр D: 100 мм
- Ширина B: 21.0 мм

**Совместимость по ISO 15**:
- Серия 7xxx: размеры могут совпадать с серией 62xx по ISO 15, но конструкция принципиально отличается
- НЕ взаимозаменяем с 6211 — другой тип нагрузки и требования к монтажу
- Парные установки: O-образная (DB), X-образная (DF), тандем (DT)

**Посадочные требования**:
- Вал: j5 или k5 (плотная посадка для точных применений)
- Корпус: H6 или J6 (для неподвижного наружного кольца)
- Осевой преднатяг: обязателен (пружиной или гайкой)
- Направление осевой нагрузки: от широкого торца наружного кольца

## Грузоподъёмность и ресурс

**Динамическая грузоподъёмность** (C): 43.6 кН (≈ 4446 кгс)
**Статическая грузоподъёмность** (C0): 32.5 кН (≈ 3314 кгс)
**Предельные обороты**: 4800 об/мин (с уплотнениями; открытый: 6000 об/мин)

**Расчёт ресурса** (по ISO 281):

Для шариковых подшипников:
```
L10 = (C / P)³ × 10⁶ оборотов
L10h = L10 / (60 × n)
P = X × Fr + Y × Fa  (для комбинированной нагрузки)
L10h = L10 / (60 × n)
```

**Пример расчёта для комбинированной нагрузки**:
- Радиальная нагрузка Fr = 2200 Н
- Осевая нагрузка Fa = 1100 Н
- Частота вращения n = 1500 об/мин
- X = 0.57, Y = 0.93 (для угла 40°, Fa/Fr > e)
- P = 0.57 × 2200 + 0.93 × 1100 = 2277 Н
- L10 = (C / P)³ × 10⁶ оборотов
- L10h = L10 / (60 × n)

**Реальный ресурс** зависит от:
- Качества смазки и условий эксплуатации
- Точности монтажа и центровки
- Загрязнённости среды
- Температурного режима

## Эквиваленты и аналоги

**Точные эквиваленты** (совпадают размеры, угол контакта):

| Производитель | Обозначение | Примечания |
|---------------|-------------|------------|
| SKF | 7211 BEP | Премиум качество, Heavy duty уплотнение |
| FAG | 7211-B-TVP | Премиум, точка вместо дефиса |
| NTN | 7211B | LLU = низкое трение уплотнения |
| NSK | 7211B | DDU = двойное уплотнение |
| ГОСТ | 36211 | Отечественный, код "18" = двухстороннее уплотнение |

**Функциональные аналоги** (допустимые отличия):
- **7211-2RS C3**: увеличенный зазор — для работы при повышенных температурах
- **7211-2Z**: металлические шайбы вместо резины — меньшая защита, выше обороты
- **7211** (открытый): нет уплотнений — требуется внешняя защита, выше обороты

**Внимание**: проверяйте совпадение суффиксов (зазор, уплотнение) при замене между производителями.

## Риски замены и ограничения

**Нельзя заменять на**:
- 7210-2RS — другой d (50 мм вместо 55 мм) — не сядет на вал
- 7212-2RS — другой d (60 мм вместо 55 мм) — не сядет на вал

**Допустимые замены с оговорками**:
- **Открытый → уплотнённый**: возможно, но снизятся обороты
- **Уплотнённый → открытый**: возможно, но нужна внешняя защита от грязи
- **CN → C3**: допустимо (C3 более универсален)
- **2RS → 2Z**: допустимо, но снижается защита от влаги

**Принципиальные отличия от радиальных подшипников**:
- Требуется осевой преднатяг — без него подшипник будет работать некорректно
- Осевая нагрузка воспринимается только в одном направлении
- Для двухсторонней осевой нагрузки — парная установка (DB или DF)
- Монтаж сложнее: необходимо соблюдать направление и величину преднатяга

**Типовые ошибки**:
- Монтаж без осевого преднатяга → повышенный люфт, вибрации, снижение ресурса
- Неправильное направление установки → осевая нагрузка не воспринимается
- Замена на радиальный подшипник 6211 → отказ при осевых нагрузках
- Чрезмерный преднатяг → перегрев, ускоренный износ

## Эксплуатация и обслуживание

**Рекомендуемая смазка**:
- Тип: литиевая (Li) консистентная, NLGI 2
- Подшипник поставляется с заводской смазкой
- Температурный диапазон смазки: –30…+120 °C
- Смазка заложена на весь срок службы (необслуживаемый)

**Периодичность замены смазки**:
- Для уплотнённых подшипников (2RS) смазка не обновляется
- При появлении признаков износа — замена подшипника целиком

**Монтаж**:
- Метод: холодный монтаж с оправкой (для малых партий)
- Горячая посадка при 80–100 °C (для серийного производства)
- Обязательно: усилие через торец внутреннего кольца (вращающееся)
- Запрещено: удары молотком, передача усилия через тела качения

**Признаки необходимости замены**:
- Повышенный шум (гул, скрежет)
- Рост температуры выше 80 °C
- Вибрации и биение шпинделя
- Потеря преднатяга (увеличение осевого люфта)
- Ухудшение точности обработки (для станков)

**См. подробнее**: [Эксплуатация и обслуживание](../../maintenance/README.md)

## Примеры применения

**Пример 1**: Шпиндель фрезерного станка (парная установка DB)
- Условия: Fr = 500 Н, Fa = 300 Н, 4000 об/мин, масляная смазка
- Ресурс: 20000–30000 часов (расчётный)
- Особенности: парная установка DB, преднатяг пружиной, масляный туман

**Пример 2**: Червячный редуктор (входной вал)
- Условия: Fr = 800 Н, Fa = 600 Н, 1500 об/мин, консистентная смазка
- Ресурс: 30000–40000 часов
- Особенности: значительная осевая от червячной передачи, преднатяг гайкой
