# Типы подшипников — классификация

Систематизация типов подшипников по конструкции, виду воспринимаемой нагрузки и области применения.

## Контекст применения

- Выбор типа подшипника исходя из нагрузок и условий эксплуатации
- Понимание конструктивных отличий и ограничений
- Определение взаимозаменяемости разных типов

## Основные классы подшипников

### По виду тел качения

1. **Шариковые** — тела качения сферической формы
   - Низкое трение, высокие обороты
   - Ограниченная грузоподъёмность
   - Примеры: радиальные однорядные (6xxx), упорные

2. **Роликовые** — тела качения цилиндрической, конической или бочкообразной формы
   - Высокая грузоподъёмность
   - Ниже предельные обороты
   - Примеры: цилиндрические (Nxxx), конические (3xxxx), сферические (2xxxx)

### По направлению воспринимаемой нагрузки

1. **Радиальные** — воспринимают радиальную нагрузку (перпендикулярно оси вращения)
   - Примеры: 6xxx (шариковые радиальные однорядные)

2. **Упорные** — воспринимают осевую нагрузку (вдоль оси вращения)
   - Примеры: 511xx (упорные шариковые)

3. **Радиально-упорные** — воспринимают комбинированную нагрузку
   - Примеры: 7xxx (шариковые радиально-упорные), конические роликовые

### По количеству рядов тел качения

- Однорядные — один ряд тел качения
- Двухрядные — два ряда (увеличенная грузоподъёмность)
- Многорядные — три и более ряда (специальные применения)

## Типовые серии и обозначения

| Серия | Тип | Применение |
|-------|-----|------------|
| 6xxx | Шариковые радиальные однорядные | Электродвигатели, редукторы, насосы |
| 7xxx | Шариковые радиально-упорные | Шпиндели, коробки передач |
| Nxxx | Роликовые цилиндрические | Тяжёлые нагрузки, низкие обороты |
| 2xxxx | Роликовые сферические | Перекосы осей, тяжёлые условия |
| 3xxxx | Роликовые конические | Колёсные узлы, редукторы |

## Алгоритм выбора типа

1. Определите вид нагрузки (радиальная/осевая/комбинированная)
2. Оцените величину нагрузки и частоту вращения
3. Учтите условия эксплуатации (температура, загрязнение, вибрации)
4. Проверьте требования к точности вращения
5. Выберите тип подшипника по критериям из ISO 281

## Примеры

- **Электродвигатель общепромышленный**: шариковые радиальные 6xxx (низкое трение, высокие обороты)
- **Редуктор с осевой нагрузкой**: конические роликовые 3xxxx (комбинированная нагрузка)
- **Вибросито с перекосами**: сферические роликовые 2xxxx (компенсация несоосности)

## Частые ошибки

- Применение шариковых подшипников при тяжёлых нагрузках — приводит к преждевременному разрушению
- Использование роликовых подшипников на высоких оборотах без учёта теплового режима
- Игнорирование требований к соосности для жёстких типов подшипников
