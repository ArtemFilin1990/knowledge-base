# Размеры и серии подшипников

Справочник по габаритным размерам, сериям и типоразмерам подшипников согласно ISO 15.

## Контекст применения

- Проверка совместимости подшипников по габаритам
- Подбор подшипника по посадочным размерам (d, D, B)
- Поиск взаимозаменяемых подшипников в рамках одной серии
- Расчёт посадок и допусков

## Основные габаритные размеры

### Стандартные обозначения (ISO 15)

- **d** — внутренний диаметр (диаметр отверстия)
- **D** — наружный диаметр
- **B** — ширина подшипника (для радиальных)
- **T** — высота подшипника (для упорных)
- **C** — ширина внутреннего кольца

### Единицы измерения

- Все размеры в миллиметрах (мм)
- Точность до 0.001 мм для прецизионных подшипников

## Серии подшипников

Серия определяет соотношение габаритов при одинаковом внутреннем диаметре:

### Серии диаметра

- **Лёгкая серия** (8, 9) — тонкое наружное кольцо, малая грузоподъёмность
- **Средняя серия** (0, 1, 2, 3) — стандартные пропорции
- **Тяжёлая серия** (4) — толстое наружное кольцо, высокая грузоподъёмность

### Серии ширины

- **Узкая** (8, 9)
- **Нормальная** (0)
- **Средняя** (1, 2, 3)
- **Широкая** (4, 5)

## Взаимозаменяемость по размерам

### Правила замены

1. **Точная замена**: совпадают d, D, B
2. **Допустимая замена**: d и D совпадают, B отличается в пределах допуска
3. **Недопустимая замена**: отличаются d или D

### Ограничения

- Проверяйте посадочные места (вал/корпус)
- Учитывайте осевое пространство для широких серий
- Для прецизионных применений — только точная замена

## Алгоритм подбора по размерам

1. Измерьте посадочные размеры (вал и корпус)
2. Определите требуемый внутренний диаметр d
3. Проверьте доступное пространство для D и B
4. Выберите серию по требованиям к грузоподъёмности
5. Проверьте совместимость по ISO 15

## Примеры

- **Вал Ø25 мм, корпус Ø62 мм**: подходит 6305 (d=25, D=62, B=17)
- **Вал Ø25 мм, ограничение по ширине <15 мм**: рассмотреть 6205 (лёгкая серия)
- **Замена 6305 на 6205**: НЕДОПУСТИМО (D отличается: 62 vs 52)

## Частые ошибки

- Замена подшипника с другим D — не войдёт в посадочное место корпуса
- Игнорирование B — подшипник не помещается в осевом направлении
- Неучёт допусков на посадку — заклинивание или проворачивание колец
