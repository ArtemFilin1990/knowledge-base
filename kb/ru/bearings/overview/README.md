# Обзор базы знаний по подшипникам

База знаний по подшипникам — единый источник технической информации для инженеров, закупки и продаж.

## Контекст применения

- Используется для выбора подшипников по эксплуатационным параметрам
- Помогает расшифровать маркировку и найти аналоги
- Содержит справочные данные по размерам, стандартам и производителям
- Предоставляет алгоритмы подбора и правила эксплуатации

## Структура базы знаний

### Классификация и конструкция
- **types/** — типы подшипников, конструктивные элементы, материалы
- **designation/** — правила маркировки, расшифровка обозначений
- **manufacturers/** — особенности обозначений производителей

### Размеры и стандарты
- **dimensions/** — серии, типоразмеры, габаритные таблицы
- **standards/** — ISO 15, ISO 281, ГОСТ 520-2011 и другие стандарты

### Применение и эксплуатация
- **selection/** — алгоритмы выбора, чек-листы, калькуляции ресурса
- **maintenance/** — смазка, монтаж, демонтаж, хранение
- **failure-analysis/** — типовые дефекты, причины отказов, диагностика

### Данные и источники
- **datasets/** — описание структурированных данных (CSV, JSON, схемы)
- **sources/** — оригинальные каталоги, прайсы, справочники

## Как пользоваться

1. **Для выбора подшипника**: начните с раздела selection/, определите требования (нагрузки, обороты, среда)
2. **Для расшифровки маркировки**: используйте раздел designation/
3. **Для поиска аналога**: смотрите карточки конкретных подшипников и таблицы эквивалентности
4. **Для проверки размеров**: используйте раздел dimensions/ и стандарты ISO 15

## Принципы работы базы

- **1 подшипник = 1 папка = 1 README.md** — атомарность статей
- **Метаданные в YAML** — структурированная навигация и поиск
- **Опора на стандарты** — ISO 15 (размеры), ISO 281 (ресурс), ГОСТ 520-2011
- **Проверяемость** — все данные со ссылками на источники

## Примеры

- Выбор подшипника для электродвигателя: см. selection/motor-bearings/
- Расшифровка обозначения "6305-2RS C3": см. designation/suffix-guide/
- Поиск аналога SKF 6305: см. карточку подшипника 6305

## Частые ошибки

- Замена подшипника без учёта зазора (C3/C4) — может привести к заклиниванию или преждевременному износу
- Игнорирование типа уплотнения (2RS vs открытый) — влияет на ресурс в загрязнённой среде
- Использование несертифицированного аналога без проверки параметров
