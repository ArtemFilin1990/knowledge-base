# Наборы данных и справочники по подшипникам

Описание структурированных данных, таблиц и справочников, используемых в базе знаний.

## Контекст применения

- Понимание структуры данных по подшипникам
- Правила обновления справочников
- Валидация и проверка данных
- Интеграция с внешними системами

## Типы наборов данных

### 1. Каталоги подшипников

**Назначение**: полный перечень типоразмеров с характеристиками

**Структура**:
- Обозначение (designation)
- Размеры (d, D, B)
- Грузоподъёмность (C, C0)
- Масса
- Предельные обороты
- Производитель

**Формат**: CSV, JSON

**Источник**: каталоги производителей (SKF, FAG, NTN, и др.)

**Обновление**: при выходе новых каталогов (обычно ежегодно)

### 2. Таблицы эквивалентности и аналогов

**Назначение**: соответствие обозначений между производителями

**Структура**:
- Базовое обозначение (ISO)
- Эквиваленты по производителям (SKF, FAG, NTN, NSK, ГОСТ)
- Условия эквивалентности (точная/допустимая)
- Примечания (отличия в зазоре, уплотнениях)

**Формат**: CSV

**Источник**: кросс-справочники производителей

**Примечания**: проверяйте совпадение зазоров и уплотнений

### 3. Справочники производителей

**Назначение**: информация о брендах и заводах

**Структура**:
- Наименование производителя
- Страна происхождения
- Особенности маркировки
- Сегмент (премиум/средний/бюджет)
- Контактная информация

**Формат**: CSV, JSON

### 4. Размерные таблицы по сериям

**Назначение**: габаритные размеры по ISO 15

**Структура**:
- Серия (62xx, 63xx, 64xx, и т.д.)
- Код диаметра (00, 01, 02, ..., 96)
- d, D, B (мм)

**Формат**: CSV

**Источник**: ISO 15:2017

### 5. Прайс-листы

**Назначение**: актуальные цены на подшипники

**Структура**:
- Обозначение
- Производитель
- Цена (валюта, НДС)
- Наличие на складе
- Срок поставки
- Минимальная партия
- Дата актуальности

**Формат**: CSV, XLSX

**Источник**: прайсы поставщиков

**Обновление**: ежемесячно или по мере поступления

**Особенности**: см. раздел [Прайс-листы](../../bearings-price-list/README.md)

## Схемы данных

### Схема каталога подшипников (catalog.csv)

```csv
designation,type,d_mm,D_mm,B_mm,C_kN,C0_kN,mass_kg,rpm_limit,manufacturer,source
6305,ball_radial,25,62,17,22.5,11.2,0.13,18000,SKF,catalog_2025
6305-2RS,ball_radial,25,62,17,22.5,11.2,0.15,10000,SKF,catalog_2025
```

**Поля**:
- `designation` — обозначение подшипника
- `type` — тип (ball_radial, roller_cylindrical, и т.д.)
- `d_mm`, `D_mm`, `B_mm` — размеры в мм
- `C_kN` — динамическая грузоподъёмность (кН)
- `C0_kN` — статическая грузоподъёмность (кН)
- `mass_kg` — масса (кг)
- `rpm_limit` — предельные обороты (об/мин)
- `manufacturer` — производитель
- `source` — источник данных

### Схема таблицы эквивалентности (equivalents.csv)

```csv
base_designation,SKF,FAG,NTN,NSK,GOST,equivalence_type,notes
6305,6305,6305,6305,6305,305,exact,
6305-2RS,6305-2RSH,6305.2RSR,6305LLU,6305DDU,180305,functional,"Check seal type"
```

**Поля**:
- `base_designation` — базовое обозначение (ISO)
- `SKF`, `FAG`, ... — обозначения производителей
- `equivalence_type` — тип эквивалентности (exact/functional)
- `notes` — примечания

## Правила обновления данных

### Процесс обновления

1. **Получение исходных данных** — новый каталог/прайс в `_sources/`
2. **Нормализация** — приведение к единой схеме
3. **Валидация** — проверка на ошибки, дубликаты
4. **Обновление датасета** — запись в CSV/JSON
5. **Документирование** — обновление README датасета с датой и источником
6. **Версионирование** — Git commit с описанием изменений

### Правила валидации

- Обозначения уникальны в рамках производителя
- Размеры d, D, B > 0
- C и C0 > 0
- Соответствие размеров стандартным сериям (ISO 15)
- Формат даты: YYYY-MM-DD

### 6. Номенклатура производителей

**Назначение**: кросс-справочник обозначений подшипников (ГОСТ ↔ международные аналоги)

**Структура**:
- Обозначение ГОСТ / российское
- Международный аналог
- Производитель / завод
- Источник данных

**Формат**: CSV

**Источник**: https://aprom.by/cgi-bin/nomenclature.pl (85 производителей, 82 951 запись)

**Файлы**:
- `nomenclature.csv` — полная таблица кросс-ссылок
- `manufacturers.csv` — справочник производителей со статистикой

**Обновление**: по мере поступления новых данных в inbox/

## Примеры использования

### Пример 1: Поиск подшипника по размерам

```python
import pandas as pd

catalog = pd.read_csv('catalogs/skf-2025.csv')
result = catalog[(catalog['d_mm'] == 25) & (catalog['D_mm'] == 62)]
print(result[['designation', 'C_kN', 'rpm_limit']])
```

### Пример 2: Поиск аналога

```python
equivalents = pd.read_csv('equivalents/cross-reference.csv')
analog = equivalents[equivalents['SKF'] == '6305-2RSH']
print(f"NTN: {analog['NTN'].values[0]}")
print(f"FAG: {analog['FAG'].values[0]}")
```

## Частые ошибки

- Устаревшие данные без указания даты — невозможно проверить актуальность
- Смешивание данных из разных источников без нормализации — противоречия
- Отсутствие валидации — ошибки в размерах и характеристиках
