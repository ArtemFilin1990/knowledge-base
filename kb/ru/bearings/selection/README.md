# Выбор подшипника — алгоритмы и методики

Практическое руководство по выбору подшипника на основе эксплуатационных параметров и расчёта ресурса.

## Контекст применения

- Подбор подшипника для нового узла
- Замена изношенного подшипника с учётом условий
- Расчёт ресурса и проверка грузоподъёмности
- Оптимизация по стоимости и надёжности

## Входные параметры для выбора

### Геометрические ограничения
- Диаметр вала (d)
- Диаметр посадочного места в корпусе (D)
- Доступная ширина (B)

### Нагрузки
- Радиальная нагрузка Fr (Н)
- Осевая нагрузка Fa (Н)
- Характер нагрузки (постоянная/переменная/ударная)

### Режим работы
- Частота вращения n (об/мин)
- Требуемый ресурс L (часы)
- Режим работы (непрерывный/периодический)

### Условия эксплуатации
- Температура окружающей среды (°C)
- Наличие загрязнений (пыль, вода, химикаты)
- Вибрации и удары
- Требования к уровню шума

## Алгоритм выбора (основной)

### Шаг 1: Определение типа подшипника

По виду нагрузки:
- **Только радиальная** (Fa = 0) → радиальные шариковые (6xxx) или роликовые (Nxxx)
- **Радиальная + малая осевая** (Fa < 0.5×Fr) → радиальные шариковые (6xxx)
- **Радиальная + значительная осевая** → радиально-упорные (7xxx) или конические (3xxxx)
- **Только осевая** → упорные (51xxx)

### Шаг 2: Выбор серии по грузоподъёмности

Для радиальных:
- **Лёгкая нагрузка** (Fr < 0.1×C) → лёгкая серия (62xx)
- **Средняя нагрузка** → средняя серия (63xx)
- **Тяжёлая нагрузка** → тяжёлая серия (64xx) или роликовые

### Шаг 3: Расчёт требуемой грузоподъёмности

По ISO 281:

**Для шариковых подшипников**:
```
L10 = (C / P)³ * 10⁶ оборотов
C = P * (L10 / 10⁶)^(1/3)
```

**Для роликовых подшипников**:
```
L10 = (C / P)^(10/3) * 10⁶ оборотов
C = P * (L10 / 10⁶)^(3/10)
```

Где:
- **L10** — номинальный ресурс (10⁶ оборотов)
- **C** — динамическая грузоподъёмность (Н)
- **P** — эквивалентная динамическая нагрузка (Н)

**Перевод в часы**:
```
L10h = L10 / (60 * n)
```
Где n — частота вращения (об/мин)

### Шаг 4: Расчёт эквивалентной нагрузки P

**Для радиальных подшипников**:
```
P = X * Fr + Y * Fa
```

Где:
- **X** — коэффициент радиальной нагрузки
- **Y** — коэффициент осевой нагрузки
- Значения X и Y берутся из каталога производителя

**Упрощённо для радиальных шариковых**:
- Если Fa/Fr ≤ 0.3: P ≈ Fr
- Если Fa/Fr > 0.3: P = 0.56*Fr + Y*Fa (Y из таблиц)

### Шаг 5: Выбор дополнительных характеристик

**Внутренний зазор**:
- **CN** (Normal) — стандартные условия
- **C3** — электродвигатели, повышенная температура
- **C4** — высокие температуры, тяжёлые режимы

**Уплотнения**:
- **Открытый** — чистая среда, высокие обороты, внешнее уплотнение
- **2Z** — защита от пыли, высокие обороты
- **2RS** — защита от воды и грязи, средние обороты

**Класс точности**:
- **P0** — общепромышленное применение
- **P6** — улучшенная точность
- **P5, P4** — прецизионные применения (шпиндели)

### Шаг 6: Проверка по каталогу

1. Найдите подшипник с подходящими d, D, B
2. Проверьте, что C ≥ расчётная требуемая
3. Убедитесь, что предельные обороты выше рабочих
4. Проверьте статическую грузоподъёмность C0 (для низких оборотов)

## Примеры расчёта

### Пример 1: Электродвигатель

**Входные данные**:
- d = 25 мм (вал)
- Fr = 1000 Н
- Fa = 100 Н (малая осевая)
- n = 1500 об/мин
- Требуемый ресурс = 20000 часов

**Решение**:
1. Тип: радиальный шариковый (Fa/Fr = 0.1)
2. Серия: 6305 (средняя)
3. L10 = 20000 × 60 × 1500 = 1800×10⁶ оборотов
4. P ≈ Fr = 1000 Н (малая осевая)
5. C = 1000 × (1800)^(1/3) = 1000 × 12.2 ≈ 12200 Н
6. Из каталога: 6305 имеет C = 22500 Н ✓
7. Зазор: C3 (электродвигатель)
8. Уплотнение: 2RS (защита от пыли)

**Результат**: 6305-2RS C3

### Пример 2: Редуктор с комбинированной нагрузкой

**Входные данные**:
- d = 30 мм
- Fr = 3000 Н
- Fa = 1500 Н (значительная осевая)
- n = 500 об/мин
- Ресурс = 10000 часов

**Решение**:
1. Тип: радиально-упорный или конический (Fa/Fr = 0.5)
2. Рассмотреть 7306 или 30206 (конический)
3. Расчёт по каталогу с учётом X и Y

## Частые ошибки

- Пренебрежение расчётом ресурса по ISO 281 — приводит к преждевременным отказам
- Игнорирование осевой нагрузки в расчёте P — ошибка в выборе грузоподъёмности
- Выбор зазора CN вместо C3 для электродвигателей — заклинивание при нагреве
- Применение уплотнённых подшипников на высоких оборотах — перегрев
