# ISO 281:2007 — Динамическая грузоподъёмность и расчётный ресурс

Подробное руководство по международному стандарту ISO 281:2007, определяющему методику расчёта динамической грузоподъёмности и номинального ресурса подшипников качения.

## Контекст применения

- Расчёт ожидаемого срока службы подшипника при заданных нагрузках и частоте вращения
- Подбор подшипника по требуемому ресурсу
- Сравнение подшипников различных типоразмеров по несущей способности
- Учёт условий эксплуатации (смазка, загрязнение, температура) при оценке ресурса
- Проектирование узлов с заданной надёжностью (90%, 95%, 99%)

## Ключевые положения

### Основные определения

**Базовая динамическая грузоподъёмность (C)** — постоянная нагрузка, которую подшипник может выдержать в течение номинального ресурса 1 000 000 оборотов. Указывается в каталогах производителей в кН.

- Для радиальных подшипников: C_r — радиальная динамическая грузоподъёмность
- Для упорных подшипников: C_a — осевая динамическая грузоподъёмность

**Базовая статическая грузоподъёмность (C₀)** — статическая нагрузка, вызывающая общую остаточную деформацию тела качения и кольца в наиболее нагруженном контакте, равную 0.0001 диаметра тела качения.

- Используется для оценки допустимости статических и ударных нагрузок
- Не участвует непосредственно в расчёте ресурса

**Номинальный ресурс (L₁₀)** — ресурс, который достигнут или превышен 90% подшипников из достаточно большой группы идентичных подшипников, работающих в одинаковых условиях.

### Формула базового ресурса L₁₀

```
L₁₀ = (C / P)^p × 10⁶  [оборотов]
```

где:

| Параметр | Описание |
|---|---|
| L₁₀ | Номинальный ресурс в оборотах (90% надёжность) |
| C | Базовая динамическая грузоподъёмность, кН |
| P | Эквивалентная динамическая нагрузка, кН |
| p | Показатель степени: **3** для шариковых, **10/3** для роликовых |

Пересчёт ресурса в часы:

```
L₁₀h = L₁₀ / (60 × n)  [часов]

где n — частота вращения, об/мин
```

### Эквивалентная динамическая нагрузка P

Эквивалентная динамическая нагрузка — это условная постоянная нагрузка, которая обеспечивает тот же ресурс, что и реальная комбинация радиальной и осевой нагрузок.

**Для радиальных подшипников:**

```
P = X · Fr + Y · Fa
```

где:

| Параметр | Описание |
|---|---|
| Fr | Радиальная нагрузка, кН |
| Fa | Осевая нагрузка, кН |
| X | Коэффициент радиальной нагрузки |
| Y | Коэффициент осевой нагрузки |

Если P < Fr, то принимают P = Fr.

### Коэффициенты X и Y для основных типов подшипников

#### Шариковые радиальные однорядные (тип 6xxx)

Выбор X и Y зависит от соотношения Fa/Fr и параметра e:

| Fa / Fr | X | Y | Условие |
|---|---|---|---|
| ≤ e | 1 | 0 | Осевая нагрузка мала |
| > e | 0.56 | от 1.0 до 2.3 | Осевая нагрузка значительна |

Параметр e зависит от отношения Fa/C₀:

| Fa / C₀ | e | Y (при Fa/Fr > e) |
|---|---|---|
| 0.014 | 0.19 | 2.30 |
| 0.028 | 0.22 | 1.99 |
| 0.056 | 0.26 | 1.71 |
| 0.084 | 0.28 | 1.55 |
| 0.110 | 0.30 | 1.45 |
| 0.170 | 0.34 | 1.31 |
| 0.280 | 0.38 | 1.15 |
| 0.420 | 0.42 | 1.04 |
| 0.560 | 0.44 | 1.00 |

#### Шариковые радиально-упорные (угол контакта α)

| Угол контакта α | X (Fa/Fr > e) | Y (Fa/Fr > e) | e |
|---|---|---|---|
| 15° | 0.44 | 1.47 | 0.38 |
| 25° | 0.41 | 0.87 | 0.68 |
| 30° | 0.39 | 0.76 | 0.80 |
| 40° | 0.35 | 0.57 | 1.14 |

#### Роликовые конические однорядные

| Параметр | Типичные значения |
|---|---|
| X (при Fa/Fr ≤ e) | 1 |
| Y (при Fa/Fr ≤ e) | 0 |
| X (при Fa/Fr > e) | 0.4 |
| Y (при Fa/Fr > e) | 0.4 × ctg(α) |
| e | 1.5 × tg(α) |

### Модифицированный ресурс L_nm

ISO 281 вводит понятие модифицированного ресурса, учитывающего надёжность и условия эксплуатации:

```
L_nm = a₁ · a_ISO · L₁₀
```

где:

| Параметр | Описание |
|---|---|
| L_nm | Модифицированный ресурс при надёжности (100 − n)% |
| a₁ | Коэффициент надёжности |
| a_ISO | Коэффициент условий эксплуатации (смазка, загрязнение, нагрузка) |
| L₁₀ | Базовый номинальный ресурс |

#### Коэффициент надёжности a₁

| Надёжность, % | Обозначение ресурса | a₁ |
|---|---|---|
| 90 | L₁₀ | 1.00 |
| 95 | L₅ | 0.64 |
| 96 | L₄ | 0.55 |
| 97 | L₃ | 0.47 |
| 98 | L₂ | 0.37 |
| 99 | L₁ | 0.25 |

#### Коэффициент a_ISO

Коэффициент a_ISO учитывает:

- **Вязкость смазки** (параметр κ — отношение фактической вязкости к требуемой)
- **Уровень загрязнения** (коэффициент eC)
- **Уровень нагрузки** (отношение C_u/P, где C_u — предел усталостной нагрузки)

Упрощённые диапазоны a_ISO:

| Условия | a_ISO |
|---|---|
| Неблагоприятные (загрязнение, плохая смазка) | 0.1–0.5 |
| Нормальные (чистота, стандартная смазка) | 0.5–2.0 |
| Благоприятные (высокая чистота, оптимальная смазка) | 2.0–50 |

## Алгоритм расчёта ресурса подшипника

1. **Определите нагрузки**: Fr (радиальная) и Fa (осевая) в кН
2. **Выберите подшипник**: определите C и C₀ из каталога
3. **Вычислите отношение Fa/C₀**: для определения параметра e и коэффициентов X, Y
4. **Определите X и Y**: по таблицам в зависимости от типа подшипника и Fa/Fr
5. **Рассчитайте P**: P = X · Fr + Y · Fa (если P < Fr, то P = Fr)
6. **Рассчитайте L₁₀**: L₁₀ = (C/P)^p × 10⁶ оборотов
7. **Пересчитайте в часы**: L₁₀h = L₁₀ / (60 × n)
8. **Примените модификаторы**: L_nm = a₁ · a_ISO · L₁₀ (при необходимости)
9. **Сравните с требуемым ресурсом**: если L₁₀h < требуемого — выберите подшипник большего размера

## Примеры

### Пример 1: Расчёт ресурса шарикового подшипника 6205

**Условия**:
- Подшипник: 6205 (C = 14.8 кН, C₀ = 7.8 кН)
- Радиальная нагрузка: Fr = 2.5 кН
- Осевая нагрузка: Fa = 0.5 кН
- Частота вращения: n = 1500 об/мин

**Расчёт**:

1. Fa/C₀ = 0.5 / 7.8 = 0.064 → интерполяция: e ≈ 0.27
2. Fa/Fr = 0.5 / 2.5 = 0.2 < e=0.27 → X = 1, Y = 0
3. P = 1 × 2.5 + 0 × 0.5 = 2.5 кН
4. L₁₀ = (14.8 / 2.5)³ × 10⁶ = 5.92³ × 10⁶ = 207.5 × 10⁶ оборотов
5. L₁₀h = 207.5 × 10⁶ / (60 × 1500) = **2 306 часов**

### Пример 2: Расчёт с осевой нагрузкой (подшипник 6308)

**Условия**:
- Подшипник: 6308 (C = 41.0 кН, C₀ = 24.0 кН)
- Радиальная нагрузка: Fr = 5.0 кН
- Осевая нагрузка: Fa = 3.0 кН
- Частота вращения: n = 3000 об/мин

**Расчёт**:

1. Fa/C₀ = 3.0 / 24.0 = 0.125 → e ≈ 0.31
2. Fa/Fr = 3.0 / 5.0 = 0.6 > e=0.31 → X = 0.56, Y ≈ 1.40
3. P = 0.56 × 5.0 + 1.40 × 3.0 = 2.8 + 4.2 = 7.0 кН
4. L₁₀ = (41.0 / 7.0)³ × 10⁶ = 5.857³ × 10⁶ = 200.9 × 10⁶ оборотов
5. L₁₀h = 200.9 × 10⁶ / (60 × 3000) = **1 116 часов**

### Пример 3: Модифицированный ресурс с учётом условий

**Условия**: подшипник из Примера 1, требуется надёжность 95%, условия нормальные (a_ISO = 1.5)

**Расчёт**:

1. L₁₀ = 207.5 × 10⁶ оборотов (из Примера 1)
2. a₁ = 0.64 (для 95% надёжности)
3. a_ISO = 1.5 (нормальные условия)
4. L₅m = 0.64 × 1.5 × 207.5 × 10⁶ = 199.2 × 10⁶ оборотов
5. L₅mh = 199.2 × 10⁶ / (60 × 1500) = **2 213 часов**

> Несмотря на повышение надёжности с 90% до 95%, ресурс уменьшился незначительно благодаря благоприятному коэффициенту a_ISO.

### Пример 4: Подбор подшипника по требуемому ресурсу

**Условия**: требуется ресурс L₁₀h ≥ 20 000 часов при Fr = 3.0 кН, Fa = 0, n = 1000 об/мин

**Расчёт**:

1. Требуемый L₁₀ = 20 000 × 60 × 1000 = 1 200 × 10⁶ оборотов
2. P = Fr = 3.0 кН (осевой нагрузки нет)
3. Требуемый C = P × (L₁₀ / 10⁶)^(1/p) = 3.0 × 1200^(1/3) = 3.0 × 10.63 = 31.9 кН
4. По таблицам: 6209 (C=32.0 кН) — минимально подходит, 6210 (C=35.1 кН) — с запасом
5. Выбираем **6210** (d=50, D=90, B=20)

## Частые ошибки

- **Неправильный показатель степени p**: использование p=3 для роликовых подшипников вместо p=10/3 — занижение ресурса на 20–40%
- **Игнорирование осевой нагрузки**: расчёт P = Fr при наличии значительной Fa — завышение ресурса
- **Неправильная интерполяция X, Y**: линейная интерполяция в таблице вместо использования ближайшего значения — ошибки до 15%
- **Перепутаны C и C₀**: использование статической грузоподъёмности вместо динамической в формуле ресурса
- **Не учтены условия эксплуатации**: расчёт только L₁₀ без коэффициентов a₁ и a_ISO — прогноз не соответствует реальности
- **Использование нагрузки в неверных единицах**: смешивание Н и кН — ошибка в 10³ раз
