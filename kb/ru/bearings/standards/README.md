# Стандарты и нормативы подшипников

Обзор ключевых стандартов, регламентирующих размеры, грузоподъёмность, точность и качество подшипников.

## Контекст применения

- Выбор подшипника с учётом требований стандартов
- Проверка соответствия подшипника нормативам
- Расчёт ресурса и грузоподъёмности
- Обеспечение взаимозаменяемости

## Основные международные стандарты (ISO)

### ISO 15 — Габаритные размеры

**Назначение**: определяет стандартные размеры радиальных подшипников (d, D, B)

**Область применения**:
- Обеспечивает взаимозаменяемость между производителями
- Задаёт размерные серии и типоразмеры
- Нормирует допуски на габариты

**Ключевые положения**:
- Размерные ряды по внутреннему диаметру
- Серии диаметра и ширины
- Допуски на габаритные размеры

### ISO 281 — Динамическая грузоподъёмность и ресурс

**Назначение**: методика расчёта номинального ресурса подшипников

**Область применения**:
- Расчёт динамической грузоподъёмности (C)
- Определение номинального ресурса (L10)
- Учёт режима нагружения и смазки

**Ключевые положения**:
- Базовая динамическая грузоподъёмность C
- Эквивалентная динамическая нагрузка P
- Формула расчёта ресурса: L10 = (C/P)^p × 10^6 оборотов
- Коэффициенты для шариковых (p=3) и роликовых (p=10/3)

### ISO 492 — Допуски подшипников качения

**Назначение**: классы точности и допуски

**Классы точности**:
- **Нормальный класс (P0)** — общепромышленное применение
- **Класс P6** — повышенная точность
- **Класс P5** — прецизионное применение (шпиндели)
- **Класс P4, P2** — сверхточные (координатно-расточные станки)

### ISO 5753 — Внутренний зазор

**Назначение**: классификация радиальных зазоров

**Группы зазоров**:
- C2 — меньше нормального
- CN (Normal) — нормальный
- C3 — больше нормального (стандарт для электродвигателей)
- C4 — значительно увеличенный
- C5 — максимальный

## Российские стандарты (ГОСТ)

### ГОСТ 520-2011 — Подшипники качения общие технические условия

**Назначение**: основной стандарт для отечественных подшипников

**Область применения**:
- Технические требования к материалам
- Термообработка и твёрдость
- Контроль качества
- Маркировка и упаковка

**Связь с ISO**: гармонизирован с ISO 15, ISO 492, ISO 281

### ГОСТ 3189-89 — Подшипники шариковые и роликовые. Система условных обозначений

**Назначение**: правила маркировки подшипников по ГОСТ

**Ключевые положения**:
- Структура обозначения
- Коды типов и серий
- Суффиксы и префиксы

## Отраслевые стандарты

### DIN (Германия)

- DIN 625 — шариковые радиальные подшипники
- DIN 720 — роликовые конические подшипники

### JIS (Япония)

- JIS B 1521 — радиальные шариковые подшипники

## Алгоритм применения стандартов

1. **Выбор размера**: используйте ISO 15 или ГОСТ 520 для определения габаритов
2. **Расчёт ресурса**: применяйте ISO 281 для оценки номинального ресурса
3. **Определение точности**: выберите класс по ISO 492 исходя из требований
4. **Назначение зазора**: используйте ISO 5753 (обычно C3 для электродвигателей)
5. **Проверка качества**: ссылайтесь на ГОСТ 520-2011 для приёмки

## Примеры

- **Электродвигатель**: ISO 15 (размеры), ISO 281 (ресурс), C3 (зазор), P0 (точность)
- **Шпиндель станка**: ISO 15, ISO 281, CN (зазор), P5 или P4 (точность)
- **Редуктор**: ISO 15, ISO 281, C3 (зазор), P6 (точность)

## Частые ошибки

- Игнорирование ISO 281 при расчёте ресурса — приводит к преждевременным отказам
- Применение зазора CN вместо C3 для электродвигателей — риск заклинивания при нагреве
- Неучёт класса точности для прецизионных применений — биение и вибрации
