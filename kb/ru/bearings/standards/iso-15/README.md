# ISO 15:2017 — Радиальные подшипники. Габаритные размеры

Подробное руководство по международному стандарту ISO 15:2017, определяющему габаритные и присоединительные размеры радиальных подшипников качения.

## Контекст применения

- Проектирование узлов с радиальными подшипниками: определение посадочных размеров вала и корпуса
- Обеспечение взаимозаменяемости подшипников различных производителей (SKF, FAG, NTN, NSK и др.)
- Подбор подшипника по заданному диаметру вала и допустимым габаритам корпуса
- Стандартизация размерных рядов для каталогов и баз данных
- Гармонизация отечественных стандартов (ГОСТ 520-2011) с международными нормами

## Ключевые положения

### Назначение и область действия

ISO 15:2017 устанавливает габаритные размеры для радиальных подшипников качения:

- **d** — внутренний диаметр (диаметр отверстия), мм
- **D** — наружный диаметр, мм
- **B** — ширина внутреннего кольца, мм
- **C** — ширина наружного кольца (для некоторых типов), мм
- **r_min** — минимальный радиус закругления фаски, мм

Стандарт распространяется на однорядные и двухрядные радиальные шариковые и роликовые подшипники.

### Размерные серии

Размерная серия подшипника определяется комбинацией **серии диаметра** и **серии ширины**. Это двузначный код, где первая цифра — серия ширины, а вторая — серия диаметра.

#### Серии диаметра

Серия диаметра характеризует соотношение наружного диаметра D к внутреннему d:

| Серия диаметра | Характеристика | Типичное применение |
|---|---|---|
| **0** | Сверхлёгкая | Приборы, маломощные механизмы |
| **2** | Лёгкая | Электродвигатели, насосы, вентиляторы |
| **3** | Средняя | Редукторы, конвейеры, общее машиностроение |
| **4** | Тяжёлая | Тяжёлое оборудование, крупные редукторы |

При одном и том же внутреннем диаметре d переход от серии 0 к серии 4 увеличивает наружный диаметр D и ширину B, что повышает грузоподъёмность.

#### Серии ширины

Серия ширины обозначается цифрой 0, 1, 2, 3, 4 и определяет ширину подшипника относительно его размерной серии по диаметру:

| Серия ширины | Характеристика |
|---|---|
| **0** | Узкая (Normal для серий 62xx, 63xx) |
| **1** | Нормальная |
| **2** | Широкая |
| **3** | Особо широкая |

> **Примечание**: для однорядных шарикоподшипников серий 62xx и 63xx серия ширины — 0 (не указывается в обозначении).

### Как читать обозначение размерной серии

Обозначение подшипника по ISO содержит код размерной серии. Пример:

```
  6  2  05
  │  │  └── Код внутреннего диаметра (d = 05 × 5 = 25 мм)
  │  └───── Серия диаметра: 2 (лёгкая)
  └──────── Тип: 6 (шариковый радиальный однорядный)
```

Для подшипников с d ≥ 20 мм код внутреннего диаметра = d / 5. Исключения:

| Код | d, мм |
|---|---|
| 00 | 10 |
| 01 | 12 |
| 02 | 15 |
| 03 | 17 |

## Алгоритм подбора подшипника по ISO 15

1. **Определите диаметр вала** (d) — задаётся конструкцией
2. **Определите допустимый наружный диаметр** (D) — ограничен размерами корпуса
3. **Выберите серию диаметра** (0, 2, 3, 4) — исходя из требуемой грузоподъёмности и ограничений по габаритам
4. **Проверьте ширину** (B) — убедитесь, что ширина подшипника вписывается в конструкцию
5. **Проверьте r_min** — радиус фаски вала и корпуса должен быть ≤ r_min подшипника
6. **Подтвердите грузоподъёмность** — используя ISO 281, убедитесь, что выбранный размер обеспечивает требуемый ресурс

## Взаимозаменяемость

ISO 15 гарантирует, что подшипник одного обозначения от любого производителя имеет **одинаковые посадочные размеры** (d, D, B). Это означает:

- Подшипник 6205 от SKF, FAG, NTN, NSK и отечественных заводов имеет d=25, D=52, B=15 мм
- Допуски на эти размеры регламентированы ISO 492
- Внутренняя конструкция (число и размер тел качения, конструкция сепаратора) может отличаться

**Ограничения взаимозаменяемости**:
- Грузоподъёмность (C, C₀) может незначительно различаться у разных производителей
- Предельные частоты вращения могут отличаться
- Суффиксы (2RS, ZZ, C3 и т.д.) не стандартизированы между производителями

## Классы точности (связь с ISO 492)

ISO 15 определяет номинальные размеры, а ISO 492 — допуски на эти размеры по классам точности:

| Класс | Допуск d (мкм) для d=25 мм | Допуск D (мкм) для D=52 мм | Применение |
|---|---|---|---|
| P0 (Normal) | 0 / −8 | 0 / −11 | Общее машиностроение |
| P6 | 0 / −5 | 0 / −7 | Электродвигатели, насосы |
| P5 | 0 / −4 | 0 / −5 | Шпиндели станков |
| P4 | 0 / −3 | 0 / −4 | Прецизионные шпиндели |

## Связь с ГОСТ

| ISO | ГОСТ | Примечание |
|---|---|---|
| ISO 15:2017 | ГОСТ 520-2011 (частично) | Габаритные размеры гармонизированы |
| ISO 15 серия 62xx | ГОСТ: тип 0 (шариковый радиальный), серия 200 | Обозначение 6205 ↔ 205 (ГОСТ) |
| ISO 15 серия 63xx | ГОСТ: тип 0, серия 300 | Обозначение 6305 ↔ 305 (ГОСТ) |

Подшипник 180205 по ГОСТ = 6205-2RS по ISO (с двусторонними уплотнениями).

## Примеры

### Пример 1: Подбор подшипника для вала электродвигателя

**Условия**: вал d = 35 мм, ограничение корпуса D ≤ 80 мм, умеренная нагрузка

1. d = 35 мм → код диаметра: 35/5 = 07
2. Ограничение D ≤ 80 → серия 62xx: D = 72 мм ✓, серия 63xx: D = 80 мм ✓
3. Выбираем 6207 (D=72, B=17, C=25.5 кН) — если грузоподъёмности достаточно
4. Или 6307 (D=80, B=21, C=33.2 кН) — если нужна большая грузоподъёмность
5. Проверяем ресурс по ISO 281

### Пример 2: Определение размеров по обозначению

**Задача**: найти размеры подшипника 6312

1. Тип: 6 — шариковый радиальный однорядный
2. Серия диаметра: 3 — средняя
3. Код диаметра: 12 → d = 12 × 5 = 60 мм
4. По таблице 63xx: D = 130 мм, B = 31 мм, r_min = 2.1 мм

### Пример 3: Замена подшипника ГОСТ на ISO

**Задача**: заменить подшипник 305 (ГОСТ) на импортный

1. 305 по ГОСТ → серия 300, код диаметра 05 → d = 25 мм
2. Эквивалент ISO: 6305 (d=25, D=62, B=17)
3. Проверяем размеры — полностью совпадают
4. Выбираем суффикс: если нужны уплотнения — 6305-2RS

## Частые ошибки

- **Путаница серий диаметра**: установка 63xx вместо 62xx — наружный диаметр не войдёт в корпус
- **Игнорирование r_min**: фаска вала больше r_min подшипника — кольцо не садится до упора
- **Неучёт ширины B при замене серии**: серия 63xx шире серии 62xx, может не поместиться по ширине
- **Прямая замена ГОСТ → ISO без проверки суффиксов**: уплотнения, зазоры, материал сепаратора могут отличаться
- **Использование устаревших таблиц**: размеры в ISO 15:2017 могут отличаться от старых редакций для некоторых типоразмеров
