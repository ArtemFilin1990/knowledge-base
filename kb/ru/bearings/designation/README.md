# Маркировка подшипников — правила расшифровки

Руководство по расшифровке обозначений подшипников: базовая маркировка, суффиксы, особенности производителей.

## Контекст применения

- Расшифровка обозначений на подшипниках и в каталогах
- Определение характеристик подшипника по маркировке
- Поиск эквивалентов у разных производителей
- Проверка соответствия требованиям

## Структура обозначения

Типовое обозначение подшипника: **[Префикс] NNNN [-Суффиксы]**

### Базовое обозначение (NNNN)

Состоит из 3-5 цифр, кодирующих:
1. **Тип и серию** (первые 1-2 цифры)
2. **Внутренний диаметр** (последние 2 цифры)

**Пример**: 6305
- **6** — серия шариковых радиальных однорядных
- **3** — серия ширины (средняя)
- **05** — код внутреннего диаметра

### Расчёт внутреннего диаметра

- **Для 00-03**: специальные размеры (10, 12, 15, 17 мм)
- **Для 04 и выше**: d = код × 5 мм
  - Код 05 → d = 25 мм
  - Код 06 → d = 30 мм
  - Код 12 → d = 60 мм

### Суффиксы (дополнительные характеристики)

| Суффикс | Значение | Примечание |
|---------|----------|------------|
| 2RS, 2RSH | Уплотнение с двух сторон (резина) | Защита от загрязнений |
| ZZ, 2Z | Защитные шайбы с двух сторон (металл) | Меньшее трение, чем RS |
| C2, C3, C4 | Радиальный зазор | C3 — увеличенный зазор (стандарт для эл.двигателей) |
| P0, P6, P5 | Класс точности | P0 — нормальный, P5 — высокий |
| /C3 | Альтернативная запись зазора | То же, что C3 |

## Различия между производителями

### SKF (Швеция)
- Использует суффиксы с дефисом: 6305-2RSH
- 2RSH = уплотнение с контактом высокого качества
- /C3 = увеличенный зазор

### FAG/INA (Германия)
- 2RSR = уплотнение резиновое
- C3 без дефиса: 6305C3

### NTN/NSK (Япония)
- DDU, LLU = уплотнение (низкое трение)
- Часто используют "//" для зазора: 6305//C3

### Отечественные (ГОСТ)
- Обозначение по ГОСТ 3189-89
- Пример: 180305 (18 — открытый, 03 — серия, 05 — диаметр)

## Алгоритм расшифровки

1. Определите базовое обозначение (серию и тип)
2. Вычислите внутренний диаметр по коду
3. Расшифруйте суффиксы (уплотнения, зазор, точность)
4. Уточните особенности для конкретного производителя
5. Проверьте совместимость с требованиями

## Примеры

### Пример 1: 6305-2RS C3
- **6** — шариковый радиальный однорядный
- **3** — серия ширины (средняя, B = 17 мм)
- **05** — d = 25 мм
- **2RS** — уплотнения резиновые с двух сторон
- **C3** — увеличенный радиальный зазор

**Применение**: электродвигатель в загрязнённой среде

### Пример 2: 7210 B P4
- **7** — шариковый радиально-упорный
- **2** — лёгкая серия
- **10** — d = 50 мм
- **B** — угол контакта 40°
- **P4** — класс точности выше нормального

**Применение**: шпиндель станка

## Частые ошибки

- Путаница между 2RS (резина) и 2Z (металл) — влияет на трение и герметичность
- Игнорирование зазора C3/C4 при замене — может привести к заклиниванию
- Неучёт различий в маркировке производителей — приводит к ошибкам при заказе
