# Номенклатура производителей подшипников — справочник аналогов

## Контекст применения

- Поиск международного аналога для подшипника с российским обозначением (ГОСТ)
- Поиск ГОСТ-обозначения для импортного подшипника
- Подбор замены от альтернативного производителя
- Анализ номенклатуры конкретного завода/бренда

## Особенности данных

- **Дубликаты**: одно ГОСТ-обозначение может иметь несколько записей у разных производителей
- **Пустые аналоги**: часть записей не имеет международного аналога (только ГОСТ)
- **Пустые ГОСТ**: часть записей имеет только международное обозначение (без ГОСТ)
- **Суффиксы**: варианты исполнения (класс точности, зазор, уплотнение) дают отдельные строки
- **Нормализация производителей**: ключ производителя `10-ГПЗ` унифицирован (источники `brand=10-ГПЗ` и `brand=10`)

## Частые ошибки

- Прямая замена без проверки суффиксов — аналог может отличаться по зазору или уплотнению
- Использование устаревших обозначений — номенклатура производителей обновляется
- Игнорирование дубликатов — один подшипник может быть указан несколько раз с разными вариантами аналогов
