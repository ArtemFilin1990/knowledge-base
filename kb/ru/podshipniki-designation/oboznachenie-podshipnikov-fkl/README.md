# Обозначение подшипников FKL.


- Система обозначений подшипников FKL.
- Префикс в обозначении подшипника(слева).
- Суфикс в обозначении подшипника (справа).
- Общие обозначения подшипников FKL.
- Обозначения типов уплотнений подшипников.
- Обозначения специальной защиты поверхности подшипников.
- Обозначения температуры подшипников.
- Подшипниковые узлы типа Y .
- Обозначения типов крепления внутреннего кольца подшипников к валу.
