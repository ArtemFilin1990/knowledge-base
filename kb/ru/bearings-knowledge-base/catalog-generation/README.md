# Инструкция по созданию базы знаний о подшипниках (полная генерация каталога)

Кому и зачем: владельцам каталога и операторам данных, которым нужен единый стандарт для проверенных позиций подшипников без дублей и ошибок.

## Контекст применения
- Используется при создании и поддержке базы знаний подшипников.
- Полезно при обработке прайс-листов, каталогов производителей и сверке ГОСТ/ISO.
- Предполагается наличие исходных файлов (Excel, CSV, PDF, DOCX, ZIP) и доступа к каталогам производителей.

## Ключевые пункты
- База знаний строится на логике ГОСТ/ISO, нормализации и приоритете каталогов производителей.
- Каждая запись проходит декомпиляцию обозначений, очистку от мусора и проверку размеров.
- Аналоги ГОСТ⇄ISO допускаются только при подтверждении по справочникам и каталогам.
- Размеры и масса берутся исключительно из каталогов, без вычислений и прайсов поставщиков.

## Алгоритм / шаги
1. Импортировать все исходные файлы проекта (Excel, CSV, PDF, DOCX, ZIP).
2. Выполнить декомпиляцию:
   - ГОСТ → позиционная схема (7-значная позиция 7654321).
   - ISO → базовый номер + корректные суффиксы.
3. Очистить данные от мусора:
   - удалить скобки и смешивание ГОСТ/ISO;
   - удалить некорректные или двойные суффиксы.
4. Проверить размеры (d/D/B/H) по каталогам производителей в порядке приоритета:
   1) SKF
   2) INA/FAG
   3) NKE
   4) ISB
   5) MPZ
   6) SNR
5. Проверить массу по каталогам производителей (без расчётов).
6. Сформировать честные аналоги ГОСТ⇄ISO с опорой на справочные документы и каталоги.
7. Нормализовать запись и привести к единой таблице.

## Правила заполнения
### 1. ГОСТ (ГОСТ 3189-89)
- Используется формат 7-значной позиции (7654321).
- Обязательные шаги:
  - полностью декомпилировать ГОСТ;
  - определить d, серию, тип (позиция 4), исполнение (5–6), серию ширин (7);
  - удалить скобки, ISO-хвосты и суффиксы, не соответствующие ГОСТ.

### 2. ISO (каталоги производителей)
- ISO хранится в чистом виде: базовый номер + стандартные суффиксы.
- Двойные/кривые суффиксы удаляются.
- Если ISO ошибочно присутствует в ГОСТ — удалить из ГОСТ.

### 3. Размеры (d/D/B/H)
- Только по каталогам производителей, без данных из прайсов поставщиков.

### 4. Масса
- Только по каталогам производителей, без вычислений.

### 5. Аналоги ГОСТ⇄ISO
- Только подтверждённые аналоги.
- Примеры:
  - 36204K6 ⇄ 7204
  - 80205 ⇄ 22205
  - 180204 ⇄ 6204-2RS

### 6. Наименование
- Формируется после разборки: тип + серия + размер + исполнение.
- Пример: «Шариковый радиальный подшипник 6204 ZZ».

## Минимальный формат записи

| --- | --- | --- | --- | --- | --- | --- | --- | --- | --- | --- |

## Частые ошибки
- Ошибочная компиляция ГОСТ (самая частая причина брака).
- Неверный ISO-аналог для шариковых подшипников 6xxx/62xx/63xx.
- Разные размеры в каталогах без зафиксированного приоритета производителя.
- Мусор из прайсов поставщиков (смешение ГОСТ и ISO, случайные суффиксы).
