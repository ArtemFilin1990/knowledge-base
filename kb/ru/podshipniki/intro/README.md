# Введение в подшипники

Вводная статья о подшипниках: назначение, основные типы и области применения. Предназначена для инженеров, закупки, сервисных специалистов и продаж B2B.

## Контекст применения

- Понимание назначения и принципов работы подшипников
- Выбор типа подшипника для конкретной задачи
- Ориентация в классификации и терминологии
- Навигация по базе знаний для углубленного изучения

## Что такое подшипник

**Подшипник** — это механический элемент, обеспечивающий вращение или линейное перемещение деталей машин с минимальным трением.

### Основные функции

1. **Снижение трения** между вращающимися или движущимися частями
2. **Передача нагрузки** от вращающегося элемента на корпус
3. **Обеспечение точности** вращения и позиционирования
4. **Компенсация перекосов** (для некоторых типов)

### Области применения

Подшипники используются в:

- **Электродвигателях** — обеспечивают вращение ротора
- **Редукторах и коробках передач** — воспринимают нагрузки от шестерен
- **Колёсных узлах** — автомобили, ж/д транспорт, промышленные тележки
- **Насосах и компрессорах** — поддержка валов роторов
- **Станках** — прецизионное вращение шпинделей
- **Бытовой технике** — стиральные машины, пылесосы, вентиляторы

## Основные типы подшипников

### Подшипники качения

**Принцип работы**: между кольцами размещены тела качения (шарики или ролики), которые катятся с минимальным трением.

**Типы по телам качения**:

1. **Шариковые** (серии 6xxx, 7xxx)
   - Низкое трение, высокие обороты
   - Ограниченная грузоподъёмность
   - Примеры: 6305, 6205-2RS, 7210-B

2. **Роликовые** (серии NU, NJ, 30xxx, 2xxxx)
   - Высокая грузоподъёмность
   - Ниже предельные обороты
   - Примеры: NU206, NJ210, 30205

**Типы по направлению нагрузки**:

1. **Радиальные** — воспринимают нагрузку перпендикулярно оси вращения
2. **Упорные** — воспринимают нагрузку вдоль оси вращения
3. **Радиально-упорные** — комбинированная нагрузка (серия 7xxx, конические)

### Подшипники скольжения

**Принцип работы**: вращение происходит при скольжении поверхностей, разделенных масляной пленкой.

**Особенности**:
- Простая конструкция
- Работа в тяжелых условиях (вибрации, удары)
- Требуют постоянной смазки
- Примеры: втулки, вкладыши коленвалов

## Ключевые характеристики

При выборе подшипника учитывайте:

1. **Габариты**:
   - d — внутренний диаметр (диаметр вала)
   - D — наружный диаметр (посадочное отверстие в корпусе)
   - B — ширина подшипника

2. **Грузоподъёмность**:
   - C — динамическая грузоподъёмность (кН)
   - C₀ — статическая грузоподъёмность (кН)

3. **Предельные обороты** (об/мин)
4. **Класс точности** (обычный, повышенный, прецизионный)
5. **Внутренний зазор** (нормальный, C3, C4)
6. **Уплотнения** (открытые, Z/ZZ, RS/2RS)

## Маркировка подшипников

Обозначение подшипника состоит из:

```
6305-2RS-C3
│││├┴┬┴─┬┘
│││   │  └── Суффиксы (уплотнения, зазор)
││└── Код диаметра (05 = d=25мм)
│└─── Серия (3 = средняя)
└──── Тип (6 = шариковый радиальный)
```

Подробнее см. [Маркировка подшипников](../../bearings/designation/README.md)

## Производители

Основные мировые производители:

- **SKF** (Швеция) — премиум, полный ассортимент
- **FAG** (Германия) — прецизионные подшипники
- **NSK, NTN** (Япония) — широкий ассортимент
- **TIMKEN** (США) — конические роликовые
- **ГПЗ (Россия)** — подшипники по ГОСТ

База знаний содержит номенклатуру от **85 производителей** (82,951 запись в [nomenclature.csv](../../bearings/datasets/nomenclature.csv)).

## Примеры

### Пример 1: Электродвигатель общепромышленный

**Задача**: подшипники для электродвигателя 1500 об/мин, вал 25 мм

**Решение**: 
- Тип: шариковые радиальные (низкое трение, высокие обороты)
- Обозначение: **6305-2RS** или **6205-2RS**
- Уплотнения: 2RS (защита от пыли)
- Зазор: нормальный или C3 (при нагреве)

См. [Карточку 6305-2RS](../../bearings/cards/6305-2rs-c3/README.md)

### Пример 2: Редуктор с осевой нагрузкой

**Задача**: подшипники для червячного редуктора, комбинированная нагрузка

**Решение**:
- Тип: конические роликовые (высокая грузоподъёмность, осевая нагрузка)
- Обозначение: **30205** или **30206**
- Установка: парами (регулировка зазора)

См. [Конические роликовые](../roller-bearings/tapered-roller/README.md)

### Пример 3: Замена ГОСТ на импортный

**Задача**: найти аналог ГОСТ 180305Л

**Решение**:
- ГОСТ 180305Л = 6305-2RS (базовое обозначение)
- SKF: 6305-2RSH
- FAG: 6305.2RSR
- NSK: 6305DDU

См. [nomenclature.csv](../../bearings/datasets/nomenclature.csv) — полный кросс-справочник

## Частые ошибки

- **Замена без учёта зазора** (C3/C4) — может привести к заклиниванию при нагреве
- **Игнорирование типа уплотнения** — открытые подшипники быстро выходят из строя в пыльной среде
- **Неправильный монтаж** — запрессовка с перекосом разрушает дорожки качения
- **Использование несертифицированных аналогов** — отличия в грузоподъёмности и ресурсе
