# Классификация подшипников

Систематизация подшипников по конструкции, виду воспринимаемой нагрузки и типу тел качения.

## Контекст применения

- Выбор типа подшипника исходя из условий эксплуатации
- Понимание конструктивных отличий и ограничений каждого типа
- Определение взаимозаменяемости разных типов
- Навигация по каталогам производителей

## Основные принципы классификации

Подшипники классифицируются по нескольким критериям:

1. **По принципу работы** — качения или скольжения
2. **По виду тел качения** — шариковые или роликовые
3. **По направлению нагрузки** — радиальные, упорные, радиально-упорные
4. **По количеству рядов** — однорядные, двухрядные, многорядные
5. **По функциональным особенностям** — самоустанавливающиеся, игольчатые, линейные

## Классификация по принципу работы

### Подшипники качения

Между кольцами размещены тела качения (шарики или ролики), обеспечивающие вращение с минимальным трением за счет качения.

**Преимущества**:
- Низкое трение
- Малый пусковой момент
- Не требуют постоянной подачи смазки
- Стандартизированы (взаимозаменяемы)

**Недостатки**:
- Чувствительны к ударам и вибрациям
- Ограниченный ресурс при высоких нагрузках
- Шумность на высоких оборотах

### Подшипники скольжения

Вращение происходит при скольжении поверхностей, разделенных масляной пленкой.

**Преимущества**:
- Простая конструкция
- Работа в тяжелых условиях (вибрации, удары)
- Бесшумность
- Компактность

**Недостатки**:
- Высокое трение при запуске
- Требуют постоянной подачи смазки
- Менее стандартизированы

См. детали: [Подшипники скольжения](../plain/README.md)

## Классификация по виду тел качения

### Шариковые подшипники

Тела качения — сферы (шарики).

**Характеристики**:
- Контакт «точечный» (на самом деле — малое пятно контакта)
- Низкое трение → высокие обороты
- Ограниченная грузоподъёмность
- Низкая чувствительность к перекосам

**Основные серии**: 6xxx, 7xxx, 51xxx

См. детали: [Шариковые подшипники](../ball-bearings/README.md)

### Роликовые подшипники

Тела качения — цилиндры, конусы или бочки.

**Характеристики**:
- Контакт «линейный» → большая площадь контакта
- Высокая грузоподъёмность
- Ниже предельные обороты
- Некоторые типы компенсируют перекосы

**Основные серии**: Nxxx, NJxxx, 2xxxx, 3xxxx

См. детали: [Роликовые подшипники](../roller-bearings/README.md)

## Классификация по направлению нагрузки

### Радиальные подшипники

Воспринимают нагрузку, направленную перпендикулярно оси вращения.

**Примеры**:
- Шариковые радиальные однорядные (6xxx)
- Роликовые цилиндрические (NUxx, NJxx)
- Сферические роликовые (2xxxx)

**Применение**: электродвигатели, редукторы, насосы, вентиляторы

См. детали: [Радиальные и упорные подшипники](./radial-vs-thrust/README.md)

### Упорные подшипники

Воспринимают нагрузку, направленную вдоль оси вращения.

**Примеры**:
- Упорные шариковые (511xx, 512xx)
- Упорные роликовые цилиндрические
- Упорные роликовые сферические

**Применение**: вертикальные валы, опоры башенных кранов, турбины

### Радиально-упорные подшипники

Воспринимают комбинированную нагрузку (радиальную + осевую).

**Примеры**:
- Шариковые радиально-упорные (7xxx)
- Конические роликовые (3xxxx)

**Применение**: шпиндели станков, колёсные узлы, коробки передач

## Классификация по количеству рядов

### Однорядные

Один ряд тел качения.

**Характеристики**:
- Компактные
- Высокие обороты
- Умеренная грузоподъёмность

**Примеры**: 6305, NU206, 30205

### Двухрядные

Два ряда тел качения.

**Характеристики**:
- Увеличенная грузоподъёмность
- Воспринимают момент
- Ниже предельные обороты

**Примеры**: 3305 (двухрядный радиальный шариковый)

### Многорядные

Три и более рядов тел качения.

**Применение**: специальные условия (экстремальные нагрузки, прецизионные шпиндели)

## Примеры применения

### Электродвигатель общепромышленный

**Условия**: радиальная нагрузка, 1500 об/мин, средняя запылённость

**Выбор**: шариковый радиальный однорядный с уплотнениями (6305-2RS)

### Колёсный узел автомобиля

**Условия**: комбинированная нагрузка (радиальная + осевая), динамические удары

**Выбор**: конический роликовый, установка парами (30205 + 30205)

### Вертикальный насос

**Условия**: осевая нагрузка от веса ротора, радиальная от дисбаланса

**Выбор**: упорный шариковый (нижняя опора) + радиальный (верхняя опора)

### Вибросито

**Условия**: перекосы, вибрации, загрязнение

**Выбор**: сферический роликовый самоустанавливающийся (23xx)
