# Концепции и классификация подшипников

Раздел объединяет фундаментальные концепции, классификацию и описание типов подшипников. Создан для структурирования знаний согласно ТЗ и обеспечения связи с детальными статьями из kb/ru/bearings/.

## Контекст применения

- Изучение основ конструкции и принципов работы подшипников
- Понимание классификации по типам, нагрузкам и применению
- Выбор подходящего типа подшипника для конкретной задачи
- Навигация к детальным техническим статьям

## Структура раздела

Этот раздел concepts/ является связующим звеном между:
- Вводными материалами (intro/)
- Детальными техническими статьями (kb/ru/bearings/)
- Практическими руководствами (playbooks/)

### Подразделы

   - Базовые понятия и назначение

2. **Классификация подшипников** → [classification/README.md](../classification/README.md)
   - Общая классификация
   - По направлению нагрузки (радиальные/упорные)
   - По типу тел качения (шариковые/роликовые)

3. **Подшипники скольжения** → [plain/README.md](../plain/README.md)
   - Конструкция и принцип работы
   - Материалы и смазка
   - Области применения

4. **Шариковые подшипники** → [ball-bearings/README.md](../ball-bearings/README.md)
   - Радиальные шариковые (глубокий желоб)
   - Радиально-упорные шариковые
   - Самоустанавливающиеся шариковые
   - Упорные шариковые
   - Специальные типы (миниатюрные, керамические)

5. **Роликовые подшипники** → [roller-bearings/README.md](../roller-bearings/README.md)
