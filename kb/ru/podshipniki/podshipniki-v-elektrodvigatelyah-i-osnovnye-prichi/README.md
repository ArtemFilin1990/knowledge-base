# Подшипники в электродвигателях и основные причины отказов.

Источник: https://aprom.by/cgi-bin/article.pl?words=podshipniki_v_jelektrodvigateljah

### Краткие технические советы для контроля и эксплуатации подшипников в электродвигателях:


#### Для правильной смазки:


• Убедитесь, что, тип смазки соответствует требованиям; объём смазки адекватный; интервалы повторной смазки соблюдаются.

• Проверьте состояние уплотнений — при наличии загрязнений они могут быть повреждены.

• При необходимости используйте смазку с большей вязкостью.

• Избегайте избыточной смазки — она тоже может вызывать проблемы.


#### Избегайте электрической эрозии подшипника:


• Используйте заземляющие кольца.

• Применяйте изолированные или гибридные подшипники.

• Применяйте изолированные муфты;

• Подключайте синусоидальный фильтр для преобразования формы сигнала.

• Учитывайте соотношение цена/эффективность — заземление вала обычно дешевле альтернатив.


#### Уменьшайте вибрацию при транспортировке:


• зафиксируйте вал в осевом направлении металлической скобой (в форме «U»);

• предварительно нагрузите неприводной конец;

• зафиксируйте приводной конец ремнём в радиальном направлении.


#### Уменьшайте вибрацию при хранении:


• периодически поворачивайте вал, чтобы избежать ложного бринеллирования.


#### Правильный монтаж подшипника:


• Никогда не используйте молоток для монтажа — это может вызвать бринеллирование.

• Обеспечьте правильное выравнивание вала;

• Балансируйте валы;

• проверяйте натяжение приводныз ремней;

• Проводите монтаж с соблюдением всех нагрузок.

• Используйте точные приборы для измерения выравнивания и натяжения.

• Следуйте официальным инструкциям производителя по установке.


#### Применяйте достаточную нагрузку на подшипник:


• Обеспечьте наличие минимальной нагрузки на подшипник — особенно важно для цилиндрических роликовых подшипников.

• Избегайте недогрузки, так как она может привести к проскальзыванию и повреждению смазочного слоя.

• Применение подшипников с предварительным натягом помогает в некоторых случаях.
