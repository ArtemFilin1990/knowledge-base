# Как подобрать сальник или манжету по размерам

Источник: https://aprom.by/cgi-bin/article.pl?words=kak_podobrat_salnik_po_razmeram

В итоге всех измерений мы получаем размер, состоящий из трех значений -  d x D x B,

где
d - внутренний диаметр, мм.

D - наружный диаметр,  мм.

B - толщина сальника, мм.


И затем по этим трем или двум основным размерам в форме поиска ниже вы сможете найти и купить сальник.


---


## Поиск манжеты / сальника по размеру

**d** - внутренний диаметр **сальника / манжеты** в мм.
**D** - внешний диаметр сальника / манжеты  в мм.
**B** - ширина сальника / манжеты  в мм.


![Основные размеры сальника / манжеты](https://aprom.by//images/oil_seal_4.jpg)




- Внутренний и внешний диаметр сальника (d и D) можно менять местами в форме для поиска - в результатах поиска будет два варианта dxDxB и DxdxB

- В прайс-листе в обозначении сальника/манжеты может отсутствовать третий размер (ширина B). Поиск будет работать по двум значениям:
*например:*

поиск манжеты 120x150


## Дюймовые размеры сальника / манжеты.


Если сальник имеет **дюймовые размеры**.


![Искать дюймовый сальник](https://aprom.by/images/djujmovyj_salnik.jpg )


Производимые изделия могут иметь как метрическую размерность, так и дюймовую. В метрических обозначениях используют целые числа, а в дюймовых десятичные дробные числа.


**Все размеры указаны в миллиметрах**.


Для метрических изделий в обозначении размерности указывается диаметр гнезда корпуса узла уплотнения.

А для дюймовых размеров в обозначении пишется внешний диаметр самого уплотнения.

Например:


Для **метрических изделий**:
сальник  6х15х6

где:

Число «6» –диаметр вала d;

Число «15» -диаметр D гнезда узла уплотнения;

Число «6»- высота уплотнения B;


Для **дюймовой размерности** изделия:

сальник 6,35х19,05х6

где:

Число «6,35» -диаметр вала d;

Число «19,05»- внешний диаметр уплотнения;

Число «6» — высота уплотнения.


## Часто задаваемые вопросы на сайте Aprom.by по сальникам.


### Как узнать сальник по размерам?


- Для этого в форме поиска сальников в шапке сайта нужно указать внутренний диаметр "d ,мм", наружный диаметр "D, мм" и "В, мм" - ширину сальника. Результат поиска покажет все возможные варианты с заданными размерами.


![](https://aprom.by//images/b178.jpg)


### Как подобрать сальник по размерам?


- В форме поиска сальников в шапке сайта подбор сальника происходит по его размерам. Для подбора укажите размеры сальника, которые вам известны и нажмите кнопку "поиск".


### Как определить сальник по валу?


- Необходимо известный размер диаметра вала указать в поле "d, мм" - внутренний диаметр в форме поиска сальников в шапке сайта. Для более точного поиска желательно указать и остальные размеры -  наружный диаметр (посадочный диаметр сальника или манжеты)  "D, мм" и ширину сальника (В, мм)


### Как найти сальник по внутреннему и наружному диаметру?


- Нужно внутренний размер указать в поле "d ,мм", наружный диаметр написать в "D, мм", в форме поиска сальников в шапке сайта. Результат поиска покажет все возможные варианты с заданными размерами.
