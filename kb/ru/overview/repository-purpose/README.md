# Назначение репозитория

## Краткое содержание

Репозиторий `knowledge-base` — централизованная база знаний, где контент поддерживается ИИ-агентом по строгой, предсказуемой структуре.

## Контекст применения

Используется для накопления экспертных знаний, превращения разрозненных файлов в структурированные статьи и поддержки единого стандарта оформления и навигации.

## Основные пункты

1. База знаний хранит только готовые статьи в `kb/ru/`.
2. Сырые материалы загружаются в `inbox/` и обрабатываются агентом.
3. Структура репозитория фиксирована: одна статья — одна папка — один `README.md`.

## Примеры

- Пользователь загружает PDF и DOCX в `inbox/`, агент создает статью в `kb/ru/`.
- При загрузке новой партии файлов агент обновляет центральный индекс `kb/ru/INDEX.md`.
