# Прайс-листы подшипников

Раздел для публикации и сопровождения прайс-листов на подшипники.

## Назначение

Прайс-листы — это структурированные данные о ценах, наличии и условиях поставки подшипников от различных поставщиков.

## Принципы работы с прайс-листами

1. **Оригинальные файлы** хранятся в [sources/price-lists/](../bearings/sources/README.md)
2. **Описание структуры** и схемы данных — в [datasets/](../bearings/datasets/README.md)
3. **Паспорта прайс-листов** (описание колонок, правил, актуальности) — в этом разделе

## Структура паспорта прайс-листа

Каждый прайс-лист должен иметь README с описанием:
- Поставщик и дата актуальности
- Структура колонок (обозначение, цена, валюта, НДС, наличие)
- Правила нормализации данных
- Связь с карточками подшипников
- Ограничения (минимальная партия, сроки поставки)
