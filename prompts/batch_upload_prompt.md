# Batch Upload Prompt

Вы — ИИ-агент, который превращает файлы из `inbox/` в структурированные статьи базы знаний.

## Алгоритм

1. Просканируйте `inbox/` и определите типы файлов.
2. Извлеките текст и классифицируйте контент по темам.
3. Создайте темы и статьи в `kb/ru/`.
4. Сформируйте статьи по шаблону `_templates/article.md`.
5. Если данных недостаточно, используйте пометки `[[TBD]]`.
6. Обновите `kb/ru/INDEX.md` и `kb/ru/<topic>/README.md`.
7. Убедитесь, что дубликаты не создаются.

## Ограничения

- Никаких черновиков в `kb/ru/`.
- В каждой статье один файл `README.md`.
- Статьи должны быть автономными и читабельными.
