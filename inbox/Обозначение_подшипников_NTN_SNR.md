## Расшифровка обозначения высокоточных прецизионных радиально-упорные шариковых подшипников  SNR.


| уплотнения | MLE - закрытый подшипник без символа - открытый подшипник |
|---|---|
| материал | CH - керамические шарики без символа - стандартный подшипник |
| серия | 70 серия 719 серия 72 серия |
| внутренний диаметр | 00 - 10мм 01 - 12мм 02 - 15мм 03 - 17мм 04 - 20мм Свыше 04: Внутренний диаметр подшипника = номер ди... |
| угол контакта | C - 15 град H - 25 град |
| сепаратор | V - текстолитовый сепаратор G1 - текстолитовый сепаратор |
| схема установки | U - универсальный подшипник DU - универсальный дуплекс DB - "спина к спине" DF - "лицо к лицу" DT - ... |
| символ точности и преднатяга | J - точность соответствует ISO |
| значение преднатяга | 7 - легкий преднатяг 8 - средний преднатяг 9 - тяжелый преднатяг X - специальный преднатяг |
| значение класса точности | 4 - класс точности P4 4S - класс точности выше P4 2 - класс точноcти P2 |
| код смазки | D - подшипник заполнен смазкой |
