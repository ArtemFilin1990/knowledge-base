## Расшифровка обозначения высокоточных прецизионных радиально-упорных шариковых подшипников BARDEN.


**подробнее о бренде BARDEN**


| материал | C - керамические шарики без символа - стандартный подшипник |

| угол контакта | 2 - 25 град без символа - 15 град |
| серия | 1 - 70 19 - 719 2- 72 |
| внутренний диаметр | 00 - 10мм 01 - 12мм 02 - 15мм 03 - 17мм 04 - 20мм Свыше 04: Внутренний диаметр подшипника = номер ди... |
| тип | H - шпиндельный |
| сепаратор | B - текстолитовый сепаратор без символа - полиамидный |
| схема установки | D - универсальный подшипник DB - "спина к спине" DF - "лицо к лицу" DT - "тандем" |
| преднатяг | L - легкий преднатяг M - средний преднатяг H - тяжелый преднатяг |
